# Стек проекта

## Backend

- Python 3.12
- FastAPI
- SQLAlchemy
- Alembic
- PostgreSQL
- Redis
- Celery
- Pydantic / pydantic-settings
- Pytest

## Frontend

- Next.js
- React
- TypeScript
- Tailwind CSS
- shadcn/ui-подход к компонентам
- lucide-react для иконок

## Parsing / Processing

- openpyxl для XLSX
- LibreOffice для XLS-конвертации
- python-docx для DOCX
- PyMuPDF и pdfplumber для text-PDF
- pdf2image и Tesseract для OCR
- RapidFuzz для детерминированного matching

## Infrastructure

- Docker Compose
- Nginx для frontend-контейнера
- Makefile для локальных команд

## Scope

Проект MedArchive / MedPartners предназначен для импорта медицинских архивов, парсинга прайс-документов, нормализации строк, сопоставления услуг и подготовки данных для админ-проверки.
